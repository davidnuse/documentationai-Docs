## Overview

The SEO Documentation Platform provides a centralized space to organize, manage, and collaborate on all your SEO project documentation. You create structured pages, embed code examples, and use interactive components to make your guides accessible and engaging. Tailored for SEO professionals, it supports everything from technical audits to content strategies.

## Quick Start

Get up and running in minutes with these essential steps.

<Steps>
  <Step title="Create Your Space" icon="plus">
    Sign up at `https://dashboard.example.com` and create a new documentation space for your SEO project.
  </Step>
  <Step title="Add Your First Page" icon="file-text">
    Navigate to the dashboard and click "New Page". Choose the `introduction.mdx` template.
  </Step>
  <Step title="Customize Content" icon="edit-3">
    Edit the YAML frontmatter and body using the built-in editor. Add components like Cards and Steps.
  </Step>
  <Step title="Publish and Share" icon="globe">
    Preview your changes, then publish. Share the live URL with your team.
  </Step>
</Steps>

<CodeGroup tabs="JavaScript,Python">

  // Example: Fetch your docs via API
